# 概要
レビュー自動チェックツールです。  
slack上のレビュー依頼のスレッドにチェック結果を自動で送信します。  

# 導入方法
```
①カリキュラムを元にpythonを導入する（テックスタディ等で導入済みかと思われます！）  
②本リポジトリをcloneする(更新を楽にするためGithub Desktopに登録お願いします)  
③vscodeのターミナルでcloneしたアプリに移動する  
④ターミナルで「brew install python-tk@3.9」を実行する  
⑤ターミナルで「python3 -m pip install -r requirements.txt」を実行する    
```

# 使い方
**◎tc_e転職_reviews14-22のレビュー依頼をチェックする場合**  
=>`python auto_review_on_slack_tc.py`を実行。  
（10秒ごとに最新のレビュー依頼がないかチェックします） 

**◎rs_review_mentorsのレビュー依頼をチェックする場合**  
=>`python auto_review_on_slack_rs.py`を実行。  
（10秒ごとに最新のレビュー依頼がないかチェックします） 

**◎高校生のチャンネルのレビュー依頼をチェックする場合**  
=>`python auto_review_on_slack_hs_A.py`を実行。 
=>`python auto_review_on_slack_hs_B.py`を実行。 

**◎手動でレビュー依頼をチェックする場合**  
=>`main.py`を実行  

# 注意点
もしエラーになる場合は、vscode右下のpythonのバージョンをクリックしてcondaが選択されているかを確認してください。  
また、エラーがNo module named '○'の場合は、ターミナルで`pip install ○`を試していただければと思います。  
それでも解消されない場合は杉山にご連絡いただけますと幸いです。  

# 不具合を見つけた場合
該当レビュー依頼のスレッドにて杉山にメンション付きで連絡いただけますと幸いです。  
特に**しなければいけない指摘をしていない場合**は報告をいただけますと非常に助かります。  
（しなくてもよい指摘をしている場合は全て拾うとキリが無いので優先度低く見させていただいております）
